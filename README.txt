Social Engineering Telegram

Небольшая сеть Telegram‑ботов для демонстрации приёмов социальной инженерии

⸻

Описание проекта

Social Engineering Telegram — это компактный репозиторий с коллекцией Telegram‑ботов, чья единственная задача — корректно запросить у собеседника номер телефона и вывести полученную информацию в консоль. Логика работы каждого бота построена на базовых техниках социальной инженерии.

Главные особенности проекта:
	•	Минимум кода — каждый бот сосредоточен лишь на сборе номера телефона.
	•	Социальная инженерия — взаимодействие с пользователем выстроено таким образом, чтобы максимально повысить вероятность добровольного предоставления контакта.
	•	Гибкая структура — количество ботов и их тематические сценарии будут расширяться по мере необходимости.

⸻

Текущая функциональность
	1.	Запуск конкретного бота.
	2.	Ожидание входящих сообщений.
	3.	Вежливый запрос номера телефона.
	4.	Вывод полученных контактов в консоль.

⸻

Перспективы развития
	•	Добавление новых ботов с различными сценариями.
	•	Расширение тем для привлечения более широкой аудитории.
